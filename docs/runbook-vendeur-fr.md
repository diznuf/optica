# Runbook Vendeur (Optica v1)

## 1) Debut de journee

1. Se connecter avec votre compte vendeur.
2. Aller dans `Caisse journaliere`.
3. Ouvrir la caisse avec le fond initial.

## 2) Vente patient

1. Creer / verifier le patient.
2. Creer la commande patient.
3. Mettre a jour le statut selon avancement.
4. Encaisser paiement (cash/carte/virement).
5. Generer recu pour un paiement.

## 3) Fin de journee

1. Retourner dans `Caisse journaliere`.
2. Compter le cash reel.
3. Cloturer la caisse.
4. Verifier l'ecart (variance).

## 4) Limitations role vendeur

- Pas d'acces au prix d'achat.
- Pas d'acces fournisseurs/achats/dettes fournisseurs.
- Pas d'acces backup/restore.
