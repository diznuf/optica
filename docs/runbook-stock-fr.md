# Runbook Gestionnaire Stock (Optica v1)

## 1) Produits et seuils

1. Aller dans `Stock`.
2. Ajouter / modifier les produits.
3. Definir le seuil de reapprovisionnement.

## 2) Achats fournisseur

1. Creer un bon de commande (BC).
2. Confirmer BC.
3. Receptionner partiellement ou totalement.
4. Verifier le restant sur detail BC.

## 3) Finance fournisseur

1. Creer facture fournisseur (avec ou sans BC).
2. Enregistrer paiements partiels.
3. Enregistrer retours fournisseur.
4. Annuler documents seulement avec raison selon les regles.

## 4) Controle stock

1. Consulter `Mouvements de stock`.
2. Consulter `Alertes stock bas`.
3. Utiliser le ledger pour expliquer toute variation.

## 5) Limitations role gestionnaire stock

- Pas de gestion utilisateurs.
- Pas de creation commande de vente patient.
- Pas d'acces backup/restore admin.
